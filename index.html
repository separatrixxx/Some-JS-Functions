parseInt();

Преобразует строку в число, убирая знаки после числа. Когда встречает первый знак, который невозможно преобразовать в число, останавливается.

В случае, если выполнить преобразование не удалось, выводит NaN.

Примеры: '45$' -> '45', 'qwerty34' -> 'NaN', '72pt6' -> '72'

indexOf();

Ищет подстроку substr в строке str, начиная с позиции pos, и возвращает позицию, на которой располагается совпадение, либо -1 при отсутствии совпадений.

Примеры: 'Mama'.indexOf('M') -> '0', 'Mama'.indexOf('R')' -> '-1', 'Mama'.indexOf('ma')' -> '2'

Необязательный второй аргумент позволяет начать поиск с определённой позиции.

Примеры: 'MAMA'.indexOf('M', 2)' -> '2'

at();

Возвращает элемент массива с данным индексом. В качестве аргумента метод принимает положительные и отрицательные числа. При отрицательном значении отсчёт происходит с конца массива.

Примеры: '["Apple", "Orange", "Plum"].at(2)' -> 'Plum', '[1, 2, 3].at(-3)' -> '1'

В случае, если аргумент превышает количество элементов в массиве, выводит undefined.

Введите массив значений, разделённых запятой (числовые значения выделять кавычками не обязательно, символьные - обязательно, скобки ставить не нужно):

push();

Добавляет элемент в конец массива.

Примеры: '["Яблоко", "Апельсин"].push("Груша")' -> 'Яблоко, Апельсин, Груша'

pop();

Удаляет последний элемент из массива и возвращает его.

Примеры: '["Яблоко", "Апельсин", "Груша"].pop()' -> 'Груша', 'Яблоко, Апельсин'

Введите массив значений, разделённых запятой (числовые значения выделять кавычками не обязательно, символьные - обязательно, скобки ставить не нужно):

shift();

Удаляет из массива первый элемент и возвращает его.

Примеры: '["Яблоко", "Апельсин", "Груша"].shift()' -> 'Яблоко', 'Апельсин, Груша'

unshift();
