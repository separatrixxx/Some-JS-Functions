parseInt

Преобразует строку в число, убирая знаки после числа. Когда встречает первый знак, который невозможно преобразовать в число, останавливается

Примеры: '45$' -> '45', 'qwerty34 -> ':(', '72pt6' -> '72'

Возможны разные реализации, в том числе удаляющая любые знаки, не приводимые к числу, до того, как встретит число

Примеры: '45$' -> '45', 'qwerty34 -> '34', '72pt6' -> '72'

indexOf

Ищет подстроку substr в строке str, начиная с позиции pos, и возвращает позицию, на которой располагается совпадение, либо -1 при отсутствии совпадений.

Примеры: 'Mama'.indexOf('M') -> '0', 'Mama'.indexOf('R') -> '-1', 'Mama'.indexOf('ma') -> '2'

Необязательный второй аргумент позволяет начать поиск с определённой позиции.

Примеры: 'MAMA'.indexOf('M', 2) -> '2'
